# Video Assets

This directory should contain the following video assets used on the landing page:

- `goal-roadmap.mp4`
- `source-memory.mp4`
- `tutor-context.mp4`
- `autopsy.mp4`
- `review-memory.mp4`
- `daily-mission.mp4`

## Recommended Specs
- **Length**: 6–12 seconds each
- **Aspect Ratio**: 16:9
- **Audio**: Muted (no audio track required)
- **Visuals**: Dark mode recording
- **Format**: Compressed MP4
- **Privacy**: No personal information shown
- **Focus**: Show only one feature per clip
- **Resolution**: 1080p preferred
- **File Size**: Ideally under 3–5 MB each

## Fallbacks
If any of these videos are missing, the landing page will automatically show animated mock UI fallbacks for that specific feature.
