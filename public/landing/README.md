# Cognition OS Landing Page Video Assets

This folder is intended to hold local `.mp4` video assets for the Cognition OS landing page. If these files are missing, the landing page will gracefully fall back to displaying animated mock UI representations of the features.

## Required (Optional) Files

For the optimal cinematic experience, provide the following video files:

- `goal-roadmap.mp4`
- `source-memory.mp4`
- `tutor-context.mp4`
- `autopsy.mp4`
- `daily-mission.mp4`
- `review-memory.mp4`

## Recommended Video Specifications

- **Duration:** 6–12 seconds per video (they will loop automatically).
- **Audio:** Must be muted (no audio track).
- **Aspect Ratio:** 16:9.
- **Resolution:** 1080p or 1440p width.
- **Format:** Highly compressed `.mp4` optimized for web (H.264).
- **Theme:** Dark mode app recording (to match the site's dark aesthetic).
- **Privacy:** Ensure absolutely NO personal or sensitive information is visible in the recordings.

*Note: You do not need to commit these heavy assets to the repository if they exceed size limits, but placing them in this folder locally will allow the Next.js server to serve them to the landing page.*
